\begin{abstract}
Identificação de sistemas possui uma vasta aplicabilidade em inúmeros ramos da engenharia, este trabalho tem o intúito
de apresentar alguns conceitos relativos a identificação de sistemas, tanto lineares quanto não lineares, além da ideia
de referência virtual para que em conjunto com a teoria de projeto de controladores baseados em dados, apresentar uma
forma de identificação de controladores não lineares. A utilização de referência virtual para a obtenção dos sinais
necessários para a caracterização do controlador ótimo de um sistema é utilizado no método VRFT. Este método serve como
base para o desenvolvimento da proposta deste trabalho que em conjunto com a teoria de identificação de sistemas não
lineares, se propõem a unir os algoritmos de identificação de sistemas não lineares com o uso de referência virtual para
a obtenção do controlador ótimo que leva o sistema a se comportar em malha fechada, como desejado. Em especial optou-se
pela caracterização do controlador utilizando estrutura de modelos NARMAX, por esta ser uma classe bastante abrangente
no que diz respeito a quantidade de sistemas reais que podem ser descritos por este tipo de estrutura de modelo. Para a
identificação de sistemas NARMAX utilizou-se um algortmo iterativo que, baseado nos sinais de entrada e saída do
sistema, consegue estimar os parametros da classe de modelos. Utilizou-se então este algoritmo para identificar a
estrutura pré-definida de controladores onde os sinais de entrada e saída utilizados para a identificação são obtidos
por meio de referência virtual, assim como no método VRFT. Para demosntrar o funcionamento tanto do algoritmo utilizado
quanto da propósta de identificação de controladores como um todo, serão apresentados exemplos ilustrativos em situações
onde o controlador ideal consegue ser representado pela classe de modelos, e quando isso não é possível.

\end{abstract}

\begin{englishabstract}{System identificatiom, Virtual reference, nonlinear system, NARMAX models, data driven
controller design}
System identification has a wide applidcability in uncountable number of branches. This work aims to present some
concepts related to linear and nonlinear system identification, beyond the idea of virtual reference so that together
with the theory of data based controller design, show a way to identify nonlinear controllers. The virtual reference
method is used to obtain the necessary signals and are widelly used in VRFT method. 
VRFT method is used as a base of this work proposal and with the nonlinear system identification theory, proposes
to unite nonlinear system identification algorithms and virtual reference to obtain the ideal controller: that one
which makes the system behaves as desired in closed loop. It was choosen to model the controller as a NARMAX model due the
wide number of classes that can be represented by this model structure.
For NARMAX system identification we used a iterative algoritm that, based on the signal from input and output plant, can
estimate the model class parameters. 
This was then used to identify the pre defined controller structure and the signals were obtained by virtual reference.
To demonstrate the operation of the algorithm used and the proposed identification controller, illustrative examples
are presented in situations where the ideal controller can be represented by the class of models, and when it is not
possible.
\end{englishabstract}
