

Identificacao de sistemas é baseada no conjunto de dados que se tem do sistema observado. Estes dados podem ser
coletados em regime normal de operacao ou sob condices pre determinadas, onde é possivel escolher previamente
o sinal de entrada que será aplicado sobre o sistema. Isso é feito para que os dados coletados sejam o mais 
informativos possíveis. \cite{ljung}

Toda a identificacão de sistemas discutidas neste trabalho é sobre os dados coletados, estes podem ser obtidos
por meio da operacão normal do sistema ou podem ser feitos por projetos de experimentos para se consiga
trazer a máxima quantidade de informacões sobre o sistema. \cite{campestrini}

Um conjunto de dados obtidos tanto por malha aberta quanto por malha fechada pode ser descrito como em 
\ref{eq:si_data_acq}. 

\begin{equation}
z^N=\left \{  u(1), y(1), ... ,u(N), y(N) \right \}
\label{eq:si_data_acq}
\end{equation}
