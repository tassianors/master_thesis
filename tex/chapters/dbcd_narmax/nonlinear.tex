\section{Identificação de controladores não lineares utilizando referência virtual}
\label{sec:dbnarmax_nonlinear}
%===============================================================================

\cite{Guardabassi}

Em \cite{campi_savaresi2006} foi apresentado uma generalização do método VRFT para sistemas não lineares, existe
entretanto a necessidade de um filtro dependente dos dados coletados do sistema e derivar esta informação com base nos
dados de entrada. O filtro em questão tem o formato:

\begin{equation}
F=(I-MD) \left ( \frac{\partial G\left [ u \right ]}{\partial u}|_{\tilde{u}} \right ) 
\label{eq:vrft_nl_filter}
\end{equation} 

Como obter o valor de $ \frac{\partial G\left [ u \right ]}{\partial u}$ é bastatne custoso e requer bom conhecimento
dos dados de entrada da planta, optou-se por utilizar uma abordagem diferente da sugerida em
\cite{campi_savaresi2006}.

Mantém-se a idéia do método VRFT para a obtenção dos sinais de entrada do controlador. De posse destes sinais e dos
sinais utilizados para excitar a planta, é possível utilizar outros algoritmos de identificação de sistemas não lineares
para determinar o ajuste dos parametros da estrutura do controlador não linear escolhido {\it{a priori}}.

% ===============================================================================
\subsection{Identificação de controladores representados por classes NARMAX}
\label{sec:dbnarmax_nonlinear_narmax}
%===============================================================================

De posse dos dados de entrada e saída do controlador, obtidos por meio da utilização de referência virtual, optou-se por
utilizar o algoritmo proposto na seção \ref{sec:nl_si_algorithms_rationals} para ajustar os parametros do controlador
não linear caracterizado por uma classe de modelos NARMAX. Este algoritmo mostrou-se bastante eficiente para
identificação de sistemas NARMAX, racionais e polinomiais. Além do mais, estas famílias de modelos são capazes de
representar uma grande variedade de sistemas não lineares.

Neste trabalho focou-se na identificação de controladores caracterizados por modelos NARMAX, mas a idéia do uso de
referência virtual em conjunto com algoritmos idependentes para identificação de sistemas não lineares, pode ser
facilmente expandida, se assim for conveninte.
