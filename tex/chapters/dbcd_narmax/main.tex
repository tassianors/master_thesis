\chapter{Identificação de sistemas não lineares utilizando referência virtual}
\label{chapter:dbnarmax}
%===============================================================================
Na seção \ref{sec:dbnarmax_nonlinear} introduz-se a idéia de utilização de métodos de projeto de controladores baseados
em dados, juntamente com a utilziação de referência virtual para a obtenção dos sinais necessários. Esta seção utiliza o
algoritmo apresentado na seção \ref{sec:nl_si_algorithms_rationals} para identificação de sistemas não lineares que
podem ser descritos por classes de modelos NARMAX polinomiais ou racionais. Alguns exemplos serão apresentados para que
se possa avaliar a qualidade das estimativas obtidas.
