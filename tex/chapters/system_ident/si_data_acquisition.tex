\section{Conjunto de dados coletados}
\label{sec:sys_ident_data_acquisition}
%===============================================================================

Identificação de sistemas é baseada no conjunto de dados coletados do sistema
observado. Estes dados podem ser obtidos em regime normal de operação ou sob
condições pré determinadas, onde é possível escolher previamente o sinal de
entrada que será aplicado sobre o sistema. Isso é feito para que os dados
coletados sejam o mais informativos possíveis. \cite{ljung} Maiores detalhes
sobre escolha de sinais para alimentação serão abordados no capitulo sobre 
projeto de experimentos (seção (\ref{sec:si_project_experiments})).
