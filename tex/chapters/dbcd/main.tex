%===============================================================================
\chapter{Controle baseado em dados}
\label{chapter:dbcd}
%===============================================================================

Este capítulo tem como objetivo introduzir o conceito de projeto de controladores baseados em dados, introduzir alguns
dos principais metodos existentes e dar enfâse maior ao metodo {\it{Virtual reference feedback tuning}} - VRFT
que utiliza referência virtual como uma de suas bases.

Inicialmente na seção \ref{sec:dbcd_definition} será introduzido o conceito de projeto de controlaodres baseados
em dados. Alguns dos principais métodos e suas subdivisões em métodos diretos: onde não existe a necessidade de
identificação da planta, para então identificar o controlador; métodos iterativos: onde a cada conjunto de dados
coletados da planta um novo controlador é encontrado e aplicado sobre a planta até que se convirja para o valor mínimo
da função custo.

Deseja-se adicionar um enfoque ao método direto que utiliza referência virtual: VRFT (seção \ref{sec:dbcd_vrft}).
Quer-se com isso introduzir e exeplificar a utilização de referência virtual para a obtenção dos sinais necessários para
a identificação do controlador ideal para este sistema.

Ao fim, na seção \ref{sec:dbcd_conclusions}, serão apresentadas as considerações finais a respeito do que foi visto
neste capitulo.
