%===============================================================================
\chapter{Identificação de sistemas}
\label{chapter:system_identification}
%===============================================================================

Identificação de sistemas consiste em caracterizar um sistema físico em um modelo matemático.
Baseado na observação deste sistema é possível coletar informações a respeito de seu comportamento.
De posse destes dados é possível prever quais serão as próximas saídas do sistema baseado 
nos seus valores passados. Chamamos esta etapa de {\it{predição}} e baseado nesta equação, que
descreve o comportamento do sistema em todos os instantes é possível determinar um {\it{critério de 
minimização}} do erro entre estes valores estimados, e os valores reis coletados do sistema.

A fim de tornar a função preditora o mais próximo possível dos dados coletados, chega-se ao
ajuste dos parâmetros que provém o menor erro possível baseado no critério escolhido. 
Normalmente o critério escolhido é quem dá nome ao método de identificação.

Neste capitulo será também discutido algumas características da estimativa fornecida pelo método 
quando existem incertezas, ruídos, ou quando os dados coletados não são suficientes para que
tenhamos uma estimativa confiável e que consiga descrever o sistema real de forma unívoca.
